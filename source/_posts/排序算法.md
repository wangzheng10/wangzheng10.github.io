# 排序算法

## 冒泡排序

> 两两比较相邻的关键字，如果反序则交换，知道没有反序的记录为止

```c
#define MAXSIZE 10
typedef struct
{
    int r[MAXSIZE+1];
    int length;
}SqList;

void BubleSort(SqList *L)
{
    int i,j;
    for (i = 1; i < L->length; i++)
    {
        for (j = L->length-1; j >= i; j--)
        {
            if (L->r[j] > L->r[j+1])
            {
                swap(L, j, j+1);
            }
        }
    }
}

void swap(SqList *L, int i, int j)
{
    int temp = L->r[i];
    L->r[i] = L->r[j];
    L->r[j] = temp;
}
```
